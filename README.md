# @mobeuv/react-native-user-agent

Use to retrieve the UserAgent in React Native.

## Installation

```sh
npm install @mobeuv/react-native-user-agent
```

## Usage


```js
import { multiply } from '@mobeuv/react-native-user-agent';

// ...

const result = await multiply(3, 7);
```
